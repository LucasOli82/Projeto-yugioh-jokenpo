# 🃏 Yu-Gi-Oh Duel (Pedra, Papel e Tesoura com Cartas)

### Este projeto é um mini-jogo baseado em Pedra, Papel e Tesoura, mas usando cartas inspiradas em Yu-Gi-Oh.
O jogador escolhe uma carta, o computador escolhe outra aleatoriamente e o resultado é definido pelas regras de vitória.

# 🎮 Como funciona

- O jogador recebe 5 cartas viradas para baixo.

- Ao passar o mouse em cima, pode ver os detalhes da carta (nome, imagem, atributo).

- Ao clicar em uma carta, ela é revelada no campo.

- O computador escolhe uma carta aleatória e também a revela.

- O jogo calcula o resultado:

    - Win → Jogador vence.

    - Lose → Computador vence.

    - Draw → Empate.

A pontuação é atualizada a cada duelo.

É possível reiniciar a partida.
